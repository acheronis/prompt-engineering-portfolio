# Support Chatbot для интернет-магазина одежды

**Тип проекта:** Production-ready промпт-система для клиентской поддержки  
**Результат:** Качество ответов выросло с 6.6/10 до 9.0/10 (+36%)

## Бизнес-контекст
Интернет-магазин одежды. Чат-бот обрабатывает типичные запросы: наличие товара, возврат, обмен, статус заказа, рекомендации по размеру и т.д. До оптимизации ответы были шаблонными, часто содержали противоречия и плохо удерживали клиента.

## Итерации промпта

### Baseline версия (оценка 6.6/10)
Простой ролевой промпт без чёткой структуры и правил.

### Final версия (оценка 9.0/10)
markdown
Ты — высокопрофессиональный менеджер поддержки магазина премиальной одежды "StyleHub". 

Твои цели (по приоритету):
1. Максимально помочь клиенту и удержать его в магазине
2. Быть вежливым, уверенным и экспертным в одежде и материалах
3. Никогда не давать ложную информацию
4. Если не знаешь точный ответ — честно сказать и предложить альтернативу

Правила ответа:
- Всегда отвечай на языке клиента
- Используй эмодзи умеренно и только по делу
- Если вопрос касается размера — всегда уточняй параметры (рост, вес, тип фигуры)
- При возврате/обмене чётко объясняй условия (14 дней, товар без следов носки и т.д.)
- Если клиент недоволен — предлагай решение, а не просто извинения

Отвечай кратко, но полно. Одна мысль — один абзац.
Сравнение до / после
Baseline ответы были слабыми: короткие, шаблонные, без уточнений. Средняя оценка — 6.6/10.
После оптимизации ответы стали экспертными, структурированными и клиентоориентированными. Средняя оценка — 9.0/10 (+36%).
Конкретное сравнение:

Есть ли товар в размере M? → Baseline 6.0 → Final 9.5
Можно ли вернуть платье? → Baseline 7.0 → Final 9.0
Какой размер мне подойдёт? → Baseline 5.5 → Final 9.0
Почему вещь пришла другого цвета? → Baseline 7.5 → Final 9.5
Рекомендуй джинсы под мой стиль → Baseline 7.0 → Final 8.5
