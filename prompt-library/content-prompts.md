# Content Automation Toolkit

Библиотека оптимизированных промптов для SMM, SEO и email-коммуникаций. Все промпты прошли 2–3 итерации и тестирование на реальных задачах магазина одежды.

**Средняя оценка финальных версий: 8.8/10**

## Промпты в библиотеке

1. **Instagram-пост для продукта** (оценка 9/10) — продающий пост с сильным хуком и призывом.
2. **Переписывание сухого текста** (8.5/10) — превращение технического описания в живой продающий копирайтинг.
3. **SEO-описание карточки товара** (9/10) — оптимизированное описание с ключевыми словами.
4. **Email после покупки** (9.5/10) — благодарность + мягкий upsell.
5. **Email с предложением возврата/обмена** (8.5/10) — удержание клиента после негативного опыта.
6. **VK-пост** (8/10) — адаптированный пост для русскоязычной аудитории.
7. **Заголовок + мета-описание для SEO** (9/10) — кликабельный заголовок и описание.

## Использованные техники
- Role + Specificity + Constraints
- Few-shot examples в сложных кейсах
- Чёткие запреты на клише, шаблоны и лишние списки

**Production-рекомендации**:
- Temperature: 0.75–0.85
- Протестировано на Claude 3.5/4 и GPT-4o
- Легко адаптируется под другие ниши (косметика, электроника и т.д.)

**Статус:** Production-ready библиотека. Можно расширять до 20+ промптов при необходимости.
