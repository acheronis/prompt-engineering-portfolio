# Weather AI Agent — автоматический утренний брифинг

Реальный AI агент в n8n который каждое утро в 8:00
получает погоду в Ереване и присылает совет на день в Telegram.

## Архитектура

Schedule Trigger (8:00) → HTTP Request (wttr.in API) →
Code (извлечение данных) → Groq API (LLM анализ) →
Telegram (отправка сообщения)

## Компоненты

**Schedule Trigger** — запуск каждый день в 8:00
**HTTP Request** — бесплатный API погоды wttr.in без ключа
**Code** — извлечение текста погоды из ответа API
**Groq API** — модель llama-3.1-8b-instant анализирует погоду
**Telegram** — отправка готового совета пользователю

## Промпт для LLM

Weather in Yerevan: [данные погоды]. Give short advice for the day. Max 2 sentences.

## Реальный результат

Бот прислал в Telegram:
"Возьмите с собой зонт, поскольку вероятность осадков составляет 100%.
Оденьтесь теплее, поскольку температура воздуха составляет 10°C."

## Что демонстрирует проект

- Интеграция с внешним API без ключа (wttr.in)
- Передача данных между блоками в n8n workflow
- Использование Groq LLM API через HTTP Request
- Автоматическая отправка в Telegram
- Реальный работающий агент без кода

## Инструменты

- n8n — оркестрация workflow
- wttr.in — бесплатный API погоды
- Groq API — llama-3.1-8b-instant
- Telegram Bot API
