# Content Automation Toolkit

Библиотека из 7+ оптимизированных промптов для контент-маркетинга (SMM, SEO, email). Все промпты протестированы и прошли несколько итераций.

## Промпты в библиотеке

1. **Instagram-пост для продукта** — продающий пост с хуком и призывом (оценка 9/10)
2. **Переписывание сухого текста** — превращение технического описания в живой продающий текст (оценка 8.5/10)
3. **SEO-описание товара** — оптимизированное описание для карточки товара (оценка 9/10)
4. **Email после покупки** — благодарность + upsell (оценка 9.5/10)
5. **Email с предложением возврата** — мягкое возвращение клиента (оценка 8.5/10)
6. **VK-пост** — адаптированный пост для русскоязычной аудитории (оценка 8/10)
7. **Заголовок для карточки товара** — кликабельный SEO-заголовок (оценка 9/10)

## Ключевые техники
- Role + Specificity + Constraints
- Few-shot в сложных кейсах
- Чёткие запреты на клише и шаблоны

**Средняя оценка промптов:** 8.8/10

**Статус:** Production-ready. Легко масштабируется под другие ниши.
