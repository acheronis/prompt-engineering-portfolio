# Неделя 8 — Финальный проект: Система промптов для найма

## Проблема
HR тратит часы на однотипные задачи: анализ резюме, составление вопросов,
оценка кандидатов, письма отказов. Задача: автоматизировать с помощью промптов.

## Система из 4 промптов

### Промпт 1 — Анализ резюме (9/10)
Задача: структурированный вывод по резюме с обоснованием решения.
Ключевые требования: честно, мягко, коротко, с обоснованием отказа.

### Промпт 2 — Вопросы для интервью (10/10)
Задача: 5 вопросов которые бьют по слабым местам кандидата.
Ключевые требования: опираться на слабые места из анализа резюме.

### Промпт 3 — Оценка ответа кандидата (10/10)
Задача: оценить ответ по шкале 1-10 с профессиональным обоснованием.
Ключевые требования: честно, коротко, по факту.

### Промпт 4 — Письмо отказа (9/10)
Задача: вежливый отказ по email с объяснением причины.
Ключевые требования: мягко, профессионально, коротко, с пожеланием на будущее.

## Результаты

| Промпт | Оценка |
|---|---|
| Анализ резюме | 9/10 |
| Вопросы для интервью | 10/10 |
| Оценка ответа | 10/10 |
| Письмо отказа | 9/10 |
| **Среднее** | **9.5/10** |

## Вывод
Система покрывает полный цикл найма от анализа резюме до отказа.
Каждый промпт связан с предыдущим — слабые места из промпта 1
используются в промпте 2 и 3.
