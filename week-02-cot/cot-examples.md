# Неделя 2 — Chain-of-Thought (CoT)

## Что это
Просишь модель думать шаг за шагом — качество ответа растёт на сложных задачах.

## Пример — Набор веса

**Промпт A (без CoT):**
что делать не могу набрать вес?

**Промпт B (с CoT):**
что делать не могу набрать вес? Думай шаг за шагом: сначала найди возможные причины, потом предложи решения по каждой.

## Разница

Промпт A — модель сразу выдала список. Без логики.
Промпт B — модель разделила: сначала причины, потом решения по каждой. Структура чётче, ответ полезнее.

## Вывод

CoT даёт мало на простых темах. На сложных задачах с рассуждением — критичен.
Правило: чем сложнее задача, тем важнее CoT.
