# Система промптов для HR

Промпт-система покрывающая полный цикл найма — от анализа резюме до письма отказа.
Разработана для автоматизации рутинных HR-задач с сохранением качества коммуникации.

## Проблема

HR-специалист тратит часы на однотипные задачи:
- Анализ резюме и формулировка выводов
- Составление вопросов под каждого кандидата
- Оценка ответов на интервью
- Написание писем отказа

Каждая задача требует концентрации, но не требует уникального решения каждый раз.

## Решение

4 промпта покрывающих полный цикл найма.
Каждый промпт связан с предыдущим — слабые места из анализа резюме
автоматически становятся основой для вопросов интервью и оценки ответов.

## Результаты

| Промпт | Задача | Оценка |
|---|---|---|
| 1 — Анализ резюме | Структурированный вывод с обоснованием | 9/10 |
| 2 — Вопросы для интервью | 5 вопросов по слабым местам кандидата | 10/10 |
| 3 — Оценка ответа | Профессиональная оценка по шкале 1-10 | 10/10 |
| 4 — Письмо отказа | Вежливый отказ с объяснением причины | 9/10 |
| **Среднее** | | **9.5/10** |

## Ключевые решения

**Связанная цепочка промптов.**
Каждый следующий промпт использует вывод предыдущего.
Слабые места из промпта 1 → основа вопросов в промпте 2 → критерии оценки в промпте 3.

**Честность без жёсткости.**
Промпт отказа настроен на мягкую но точную формулировку причины.
Без корпоративных клише, без ложных обещаний.

**LLM-as-a-judge.**
Промпт 3 использует модель для оценки ответов кандидата.
Это реальная техника применяемая в production AI-системах.
