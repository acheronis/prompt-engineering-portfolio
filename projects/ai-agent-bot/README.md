# AI Agent — Бот поддержки клиентов

Архитектура автономного агента для обработки входящих обращений в Telegram.
Агент отвечает на вопросы клиентов по базе знаний без участия человека.

## Архитектура

Входящее сообщение → Telegram Trigger
→ Claude (RAG промпт + база знаний)
→ Проверка: есть ответ в базе?
  → Да: отправить ответ клиенту
  → Нет: эскалировать менеджеру

## Компоненты

**1. Telegram Trigger**
Получает входящее сообщение от клиента.
Передаёт текст сообщения в следующий блок.

**2. Claude (LLM блок)**
Получает сообщение + базу знаний.
Генерирует ответ строго по базе.
Если информации нет — возвращает флаг эскалации.

**3. Условный блок (If/Else)**
Проверяет флаг эскалации.
Да → отправить стандартный ответ + уведомить менеджера.
Нет → отправить ответ клиенту.

**4. Telegram Send Message**
Отправляет финальный ответ клиенту.

## Системный промпт агента

Ты ассистент поддержки интернет-магазина одежды.
Отвечай ТОЛЬКО на основе базы знаний.
Если ответа нет — верни строго: ESCALATE.
Не придумывай информацию.
Отвечай коротко и по делу. Один ответ — одно предложение.

База знаний:
---
Доставка: 1-3 рабочих дня. Бесплатно от 5000₽, иначе 299₽.
Возврат: в течение 14 дней в оригинальной упаковке.
Размеры: XS, S, M, L, XL.
Оплата: карта, СБП, наличные при получении.
---

Вопрос клиента: [текст сообщения]

## Логика эскалации

Агент возвращает ESCALATE если:
- Вопрос не в базе знаний
- Клиент упоминает жалобу или возврат денег
- Сумма превышает 5000₽

При эскалации клиент получает:
"Ваш вопрос передан менеджеру. Ответим в течение 15 минут."
