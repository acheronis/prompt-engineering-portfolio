# Content Automation Toolkit

Библиотека из 7 оптимизированных промптов для SMM, SEO и email (на основе задач магазина одежды). Средняя оценка 8.8/10.

## Промпты

1. Instagram-пост продукта (9/10)
2. Переписывание сухого текста в продающий (8.5/10)
3. SEO-описание карточки товара (9/10)
4. Email после покупки с upsell (9.5/10)
5. Email возврата/удержания клиента (8.5/10)
6. VK-пост (8/10)
7. SEO-заголовок + мета-описание (9/10)

Role + Specificity + Constraints + Few-shot.
