# Prompt Engineering Portfolio — Арсен Сергеян

Профессиональная коллекция промпт-систем и реальных кейсов. Специализация: создание надёжных многоагентных промпт-систем для клиентской поддержки и HR-процессов.

**Ключевые результаты:**
- Повысил качество ответов чат-бота поддержки магазина одежды с 6.6/10 до 9.0/10 (+36%)
- Построил связанную HR-систему полного цикла найма со средней оценкой 9.5/10

Используемые модели (2025–2026): Claude 3.5/4, GPT-4o, Grok, Gemini.
