# Prompt Engineering Portfolio — Арсен Сергеян

Профессиональная коллекция промпт-систем и кейсов. Специализация: создание надёжных, воспроизводимых промптов и многоагентных систем для поддержки клиентов, HR-процессов и контент-автоматизации.

**Ключевые результаты:**
- Повысил качество ответов чат-бота поддержки магазина одежды с 6.6/10 до 9.0/10 (+36%)
- Построил HR-систему (анализ резюме → вопросы → оценка → отказ) со средней экспертной оценкой 9.5/10
- Собрал и оптимизировал библиотеку из 20+ production-ready промптов для SMM, SEO и email

Используемые модели (2025–2026): Claude 3.5/4, GPT-4o, Grok, Gemini.

### 2. HR Prompt System
Многоагентная система для рекрутинга. Автоматизация от анализа резюме до обоснованного отказа.  
[→ Перейти к проекту](/projects/hr-prompt-system)

### 3. Content Automation Toolkit
Библиотека промптов для генерации постов, SEO-текстов и email-рассылок с итерациями и оценками.  
[→ Открыть библиотеку](/prompt-library)

## Техники и методологии
- Chain-of-Thought, ReAct, Self-Consistency
- LLM-as-a-Judge + собственный evaluation framework
- Сравнительный анализ техник (baseline / Role+Spec / Few-shot / CoT / ReAct)

## Как устроено портфолио
Каждый проект содержит:
- Бизнес-задачу и контекст
- Итерации промптов (v1 → final)
- Примеры input → output
- Метрики качества и анализ
- Production-рекомендации (temperature, безопасность, cost)
