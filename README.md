## Проекты

### 1. Support Chatbot — интернет-магазин одежды
Система промптов для 5 типов обращений в поддержку.
Рост качества: +40%. Три итерации промпта с документацией.
[→ Открыть кейс](./projects/support-chatbot/README.md)

### 2. HR Prompt System
Многоагентная система: анализ резюме → вопросы → оценка → отказ.
Средняя оценка: 9.5/10.
[→ Открыть систему](./projects/hr-system/README.md)

### 3. Structured Output — JSON извлечение
Система промптов для извлечения структурированных данных из свободного текста.
Применение: классификация обращений, маршрутизация тикетов.
[→ Открыть проект](./projects/structured-output/README.md)

### 4. Dialog System — голосовой робот клиники
Многошаговый диалоговый сценарий для записи пациентов.
Полный цикл: приветствие → специалист → время → данные → подтверждение.
[→ Открыть проект](./projects/dialog-system/README.md)

### 5. RAG Prompt System — база знаний магазина
Система промптов для RAG архитектуры.
Модель отвечает строго по базе знаний, не галлюцинирует при отсутствии данных.
[→ Открыть проект](./projects/rag-system/README.md)

### 6. Content Automation Toolkit
Библиотека из 7 промптов для SMM, SEO и email.
Средняя оценка: 9.3/10.
[→ Открыть библиотеку](./prompt-library/content-prompts.md)

### 7. AI Agent Bot — автономная поддержка в Telegram
Архитектура агента для обработки обращений без участия человека.
RAG промпт + логика эскалации + Telegram интеграция.
[→ Открыть проект](./projects/ai-agent-bot/README.md)

### 8. Prompt Comparison Framework
Систематическое сравнение 5 техник на одной задаче с метриками.
Baseline vs Role vs Few-shot vs CoT vs ReAct.
[→ Открыть проект](./projects/prompt-comparison/README.md)
